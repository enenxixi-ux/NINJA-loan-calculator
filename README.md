一個結合「極簡黑白美學」與「影之忍術特效」的專業貸款試算工具。
A professional minimalist loan calculator merging precise financial logic with ninja-themed dynamic effects.

---

## 🌟 專案特色 | Features

### 忍者新風格
- **硬派美學**：嚴格採用黑、灰、白三色設計，呈現冷靜且專業的商務質感。
- **忍術特效**：內建三種不同型態的手裡劍（飛鏢），以沉穩的速度高速射出，模擬戰場肅殺感。
- **人性化輸入**：專為在地使用者設計，「輸入 35」即代表「35 萬元」，讓操作更直覺。
- **精準本息均攤**：嚴格採用銀行級 PMT 公式，確保月付額與總利息計算 100% 準確。
- **防撞排版優化**：針對移動端（iPhone/Android）進行橫幅佈局修正，按鍵與標題絕不重疊。
- **一鍵分析截圖**：內建截圖按鍵，自動隱藏 UI 元素，生成乾淨專業的財務分析圖。

### En NINJA style Highlights
- **Minimalist Aesthetic**: High-contrast monochrome design for a sleek, professional business look.
- **Shinobi Animation**: Featuring 3 types of animated Shurikens (Ninja Stars) with controlled velocity.
- **Precision Math**: Built-in Equal Installment (PMT) algorithm for high-accuracy financial planning.
- **Smart Screenshot**: Integrated reporting tool that auto-hides UI buttons for clean image exports.
- **Responsive Layout**: Mobile-first flexbox design ensures no overlapping elements on any screen size.

## 🛠 技術棧 | Tech Stack

- **HTML5 / CSS3**: 自定義 Clip-path 繪製手裡劍形狀與 CSS 關鍵幀動畫。
- **JavaScript**: 純原生 ES6 邏輯處理複雜貸款公式（無需依賴笨重框架）。
- **html2canvas**: 實現網頁 DOM 轉 PNG 的高畫質截圖技術。
- **Google Fonts**: 使用 Inter 與 DM Mono 提升數位商務感。

---

## 🚀 如何使用 | How to Use

1. **下載 (Download)**: 將 `index.html` 下載至您的裝置。
2. **執行 (Run)**: 直接用瀏覽器開啟 `index.html` 即可使用。
3. **操作 (Operation)**:
   - 輸入貸款金額（萬元）、期數與利率。
   - 數據會隨著您的輸入即時精準跳動。
   - 按下「截圖」即可產出精美報告。
